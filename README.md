# 🎁 Secret Santa Gift Giver

A simple and elegant React application to organize gift exchanges (Secret Santa). Built with **React**, **React Router**, and **Bulma CSS**.



## 🚀 Features

* **People Management**: View a list of participants and search through them.
* **Dynamic List**: Add new participants on the fly without page reloads.
* **Smart Selection**: Select participants and track the count in real-time.
* **Fair Draw Algorithm**: Uses the Fisher-Yates shuffle to ensure nobody draws themselves.
* **Secret Reveal**: Pairs are hidden by default to prevent "peeking".
* **Responsive Design**: Fully styled with Bulma CSS, mobile-friendly.

## 🛠️ Tech Stack

* **Frontend**: React (Vite)
* **Routing**: React Router 6
* **Styling**: Bulma CSS
* **State Management**: React Context API (via Outlet Context)
* **Icons/Utilities**: Classnames
